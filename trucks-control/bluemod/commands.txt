---------------- Central -----------------
- AT&F1 (Set all parameters to factory defaults) *RODAR SOMENTE UMA VEZ

- AT+LETIO=4 (LE secure connections pairing with encryption)
- AT&W
- AT+RESET

- AT+BIOCAP=2 (Keyboard only)
- AT+BMITM=1 (Active MAN IN THE MIDDLE)
- AT+BNAME=BLE+CENTRAL %12a (Set module name)
- AT%B8 (Set baud rate to 115200bps)
- AT+LECONINTMIN=3200 (Set connection interval min)
- AT+LECONINTMAX=3200 (Set connection interval max)
- AT+LEROLE=1 (Set to central)
- AT+LESCANDURATION=120 (Set scan duration)
- AT+LESCAN (Scan for new connections)
- ATD (Connect to some peripheral)
-------------- security section ----------------
The modules will receive a "SSPPIN <BT addr>, tx ?" event and
will send the same passkey with BSSPPIN command.

AT+BSSPPIN <Bt addr>,tx,<passkey>
exemplo: AT+BSSPPIN 008025007C08D,t2,123456

After that the modules will receive:

- OK
- RING
- CONNECT connAddr
-------------- security section ----------------
AT+LESRVD=connAddr (search for services on peripheral)
AT+LEWRITE=connAddr,charHnd, value (write something in peripheral)
AT+LEWRITECMD=connAddr,charHnd, value (write something in peripheral without response)
AT+LEREAD (read something in peripheral)
